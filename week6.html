День 1. Прогрессивные личности.

<progress> - создает прогресс-бар, который мы можем анимировать через JavaScript.

Загрузка мозга: >Загрузка мозга

<meter> - отображает числовые значения в баре и в зависимости от значения окрашивает строку в определенный цвет.

Пример1 (значение=20):

- оптимальное 54

- оптимальное 14

- оптимальное 89

Пример 2 (нервное состояние):

хорошее - хорошее

волнуется - волнуется

может ударить - может ударить

Если браузер не понимает теги <progress> и <meter> вместо рисунков он отображает текст, который находится в них.

<details> - создает спойлер.

<summary> - создает название для спойлера.

День 2. Псевдоэлементы.

Псевдоэлементы позволяют использовать какие то новые элементы не добавляя в HTML код.

::after - добавляет псевдоэлемент в конце тега.

::before - добавляет псевдоэлемент в начале тега.

Псевдоэлементы ::after и ::before работают при наличии свойства content

Когда не нужен текст для псевдоэлементов можно в контенте указать пустой символ пробел или спецсимвол пробела

::first-letter - задает стили для первого символа

::first-line - задает стили для первой строки

Лучше использовать новые (два двоеточия, старые - одно) псевдоэлементы, тк они работают во всех браузерах.

::selection - позволяет задавать стили к выделенному тексту.

Пример:

Вот такой псевдоэлемент! Lorem ipsum dolor sit amet, consectetur adipisicing elit. Aliquam, ullam!

Тестовая цитата - Lorem ipsum dolor sit amet, consectetur adipisicing elit. Commodi dolores blanditiis delectus sunt, explicabo quod sapiente officiis, in ratione sequi eum. Voluptates, harum aliquid adipisci cumque corporis!

День 3. Псевдоклассы.

:first-child - задает стили для первого дочернего элемента своего родителя.

:last-child - задает стили для последнего дочернего элемента своего родителя.

:nth-child(значение) - задает стили через нумерацию дочерних элементов родителя.

Значение псевдокласса:

  • odd - применяет стили к нечетным элементам
  • even - применяет стили к четным элементам
  • число - задает значение определенному элементу
  • 1 - равносилен псевдоклассу :first-child
  • 2n - равносилен значению even
  • 2n+1 - равносилен значению odd
  • xn+y - x - это шаг, y - с какого знгачения начинать (например 3n+2). y может быть и отрицательным.

:nth-last-child(значение) - задает стили через нумерацию дочерних элементов родителя начиная с конца.

Когда мы ему задаем значение равное 1 - оно равносильно псевдоклассу :last-child

Тестовый список:

  • Lorem ipsum.
  • Dolorem, reprehenderit.
  • Iste, autem.
  • Explicabo, quam!
  • Quidem, tempora.
  • Possimus, eos.
  • Nihil, ipsa.
  • Deserunt, voluptatum.
  • Porro, facilis.
  • Voluptatem, eaque.

:nth-of-type(значение) - задает стили через нумерацию дочерних элементов родителя согласно их типу.

:nth-last-of-type(значение) - задает стили через нумерацию дочерних элементов родителя согласно их типу начиная с конца.

День 4. Псевдоуникумы...

:only-child - указывает стили для элементов которые являются единственными у родителя.

:only-of-type - указывает стили для элементов которые являются единственными своего типа у родителя.

:empty - указывает стили для элементов которые являются пустыми.

:not(селектор) - указывает стили для элементов которые не имеют указанный селектор.

Один Два Три Четыре Пять Шесть

:root - применяет стили ко всему документу. Аналогичен <html>

:target - применяет стили к якорям по которым мы переходим. При чем когда мы уходим из якорей стили пропадают.
